# jsMom

Welcome to JsMom a library to reproduce a typical mom scream

## Usage

``` js
  import jsMom from 'js-mom';

  const myAwesomeString = 'Come back until 8pm';

  console.log({ scream: jsMom(myAwesomeString) });

  // output: COME BACK UNTIL 8PM!

```
